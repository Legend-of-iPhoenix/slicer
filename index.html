How to use:

  1. Get an STL file off of thingiverse or something.
  2. Convert it to an ASCII STL file (as opposed to a binary STL file) using a website like this one. (binary STL support coming soon)
  3. Load it into the file input below. Wait a little bit, your browser will become unresponsive while the STL parses.
    • If this takes longer than a few seconds, send me a link to your STL file.
    • TO-DO: delegate this to a webworker or something so it happens asynchonously and doesn't freeze.
  4. Press "Go!"
